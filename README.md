# Dingtalk_createLike 

[![QQ Group](https://img.shields.io/badge/QQ%20Group-164725525-12B7F5?logo=tencent-qq)](https://jq.qq.com/?_wv=1027&k=OuhCQlwI)

一个钉钉点赞小工具，使用 `FusionApp` 制作，轻松上21亿

## 功能&特性
1. 支持手机端无抓包自动获取直播间UUID，*通过读取log文件实现*
2. 可自定义点赞数量
